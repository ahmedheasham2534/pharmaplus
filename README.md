# 💊 PharmaPlus — Smart Pharmacy POS System

## 📌 Overview

PharmaPlus is a complete Pharmacy Point of Sale (POS) system designed to streamline pharmacy operations, including sales, inventory management, order tracking, and reporting.

The system provides both **staff (desktop dashboard)** and **customer (mobile interface)** experiences in a single application.

---

## 🚀 Features

### 🧾 POS System

* Add products to cart
* Real-time stock validation
* Automatic tax calculation (14%)
* Multiple payment methods (Cash, Card, Insurance)

### 📦 Inventory Management

* Add, edit, and delete medications
* Low stock alerts
* Stock status indicators
* SKU & barcode support

### 📊 Orders Management

* Kanban board (Ordered → Preparing → Delivered)
* Drag & drop order updates
* Order details modal

### 🚚 Delivery Tracking

* Real-time tracking status
* Progress visualization
* Auto status simulation

### 📈 Reports & Analytics

* Sales insights
* Category performance
* Daily revenue tracking
* KPI dashboard

### 📱 Mobile Experience

* Customer shopping interface
* Cart & checkout
* Order history & tracking

### 🔐 Authentication

* Staff & Customer login system
* Role-based UI (Desktop vs Mobile)
